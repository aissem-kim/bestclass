모든 문제를 풀어본 후 아래 버튼으로 정답을 확인하세요.

Q1Blake와 같은 부서 사원단일행 =
SELECT  ename, hiredate
FROM    emp
WHERE   deptno = (SELECT deptno FROM emp WHERE ename = 'BLAKE')
  AND   ename != 'BLAKE';
Q2평균 급여 이상 사원 (급여 내림차순)단일행 ≥
SELECT  empno, ename
FROM    emp
WHERE   sal >= (SELECT AVG(sal) FROM emp)
ORDER BY sal DESC;
Q3'T' 포함 사원과 같은 부서 근무 사원 (사번순)IN
SELECT  empno, ename, sal
FROM    emp
WHERE   deptno IN (
            SELECT DISTINCT deptno FROM emp WHERE ename LIKE '%T%'
        )
ORDER BY empno;
Q4Dallas와 NEW YORK 근무 사원IN
SELECT  ename, job, sal
FROM    emp
WHERE   deptno IN (
            SELECT deptno FROM dept
            WHERE  loc IN ('DALLAS', 'NEW YORK')
        );
Q5King에게 보고하는 사원단일행 =
SELECT  ename, sal
FROM    emp
WHERE   mgr = (SELECT empno FROM emp WHERE ename = 'KING');
Q6SALES 부서 사원명과 업무단일행 =
SELECT  ename, job
FROM    emp
WHERE   deptno = (
            SELECT deptno FROM dept WHERE dname = 'SALES'
        );
Q730번 부서 최저 월급보다 많은 사원단일행 >
SELECT  *
FROM    emp
WHERE   sal > (SELECT MIN(sal) FROM emp WHERE deptno = 30);
Q810번 부서 중 30번 부서와 같은 업무IN
SELECT  ename, job
FROM    emp
WHERE   deptno = 10
  AND   job IN (
            SELECT DISTINCT job FROM emp WHERE deptno = 30
        );
Q9Ford와 업무·월급 모두 같은 사원다중컬럼
SELECT  *
FROM    emp
WHERE   (job, sal) = (
            SELECT job, sal FROM emp WHERE ename = 'FORD'
        )
  AND   ename != 'FORD';
Q10업무가 JONES와 같거나 FORD 월급 이상 (업무별, 급여 내림차순)OR 복합
SELECT  ename, job, deptno, sal
FROM    emp
WHERE   job = (SELECT job FROM emp WHERE ename = 'JONES')
   OR   sal >= (SELECT sal FROM emp WHERE ename = 'FORD')
ORDER BY job, sal DESC;
Q11SCOTT 또는 WARD와 월급이 같은 사원IN · NOT IN
SELECT  ename, job, sal
FROM    emp
WHERE   sal IN (
            SELECT sal FROM emp WHERE ename IN ('SCOTT', 'WARD')
        )
  AND   ename NOT IN ('SCOTT', 'WARD');
Q12CHICAGO 근무 사원과 같은 업무중첩 IN
SELECT  ename, job
FROM    emp
WHERE   job IN (
            SELECT DISTINCT job FROM emp
            WHERE  deptno = (SELECT deptno FROM dept WHERE loc = 'CHICAGO')
        );
Q13부서별 평균보다 높은 월급 사원상관 서브쿼리
SELECT  e.deptno, e.ename, e.sal
FROM    emp e
WHERE   e.sal > (
            SELECT AVG(sal) FROM emp
            WHERE  deptno = e.deptno
        )
ORDER BY e.deptno;
Q14업무별 평균보다 낮은 월급 사원상관 서브쿼리
SELECT  e.deptno, e.ename, e.sal
FROM    emp e
WHERE   e.sal < (
            SELECT AVG(sal) FROM emp
            WHERE  job = e.job
        )
ORDER BY e.deptno;
Q15말단 사원 (부하 직원이 없는 사원)NOT IN
SELECT  empno, ename, job, deptno
FROM    emp
WHERE   empno NOT IN (
            SELECT DISTINCT mgr FROM emp WHERE mgr IS NOT NULL
        );

모든 문제를 풀어본 후 아래 버튼으로 정답을 확인하세요.

Q1Timothy와 같은 부서 사원 (Timothy 제외)단일행 =
SELECT  last_name, hire_date
FROM    employees
WHERE   department_id = (
            SELECT department_id FROM employees
            WHERE  last_name = 'Timothy'
        )
  AND   last_name <> 'Timothy';
Q2평균 급여보다 많은 사원 (급여 오름차순)단일행 >
SELECT  employee_id, last_name
FROM    employees
WHERE   salary > (SELECT AVG(salary) FROM employees)
ORDER BY salary;
Q3이름에 'u' 포함 사원과 같은 부서 사원IN
SELECT  employee_id, last_name
FROM    employees
WHERE   department_id IN (
            SELECT department_id FROM employees
            WHERE  last_name LIKE '%u%'
        );
Q4위치 ID 1700인 부서 사원 이름, 부서번호, 업무 IDIN

Q5King에게 보고하는 사원 이름과 급여단일행 =
SELECT  last_name, salary
FROM    employees
WHERE   manager_id = (
            SELECT employee_id FROM employees
            WHERE  last_name = 'King'

Q6Executive 부서 사원 부서번호, 이름, 업무 IDIN

FROM    employees
WHERE   department_id IN (
            SELECT department_id FROM departments
            WHERE  department_name = 'Executive'
        );
Q7평균급여 초과 + 'u' 포함 사원과 같은 부서복합 IN
