Q 0
이름, 업무, 급여(sal), 보너스(comm), 급여+보너스를 출력하세요.💡 comm이 NULL이면 IFNULL 처리
Q 1
사원번호, 이름, 업무, 급여, 현재 급여에 15% 증가된 급여(New Salary), 증가액(Increase)를 출력하세요.
Q 2
이름, 입사일, 입사일로부터 현재까지의 년수, 급여, 입사일로부터 현재까지의 급여 총계를 출력하세요.
Q 3
이름, 업무, 입사일, 입사한 요일을 출력하세요.💡 DAYNAME 또는 CASE + DAYOFWEEK
Q 4
모든 사원의 이름과 급여를 출력하세요.💡 급여는 15자리, 좌측 빈 곳에 * 대치

모든 문제를 풀어본 후 아래 버튼으로 정답을 확인하세요.

Q0이름, 업무, 급여, 보너스, 급여+보너스
SELECT  ename,
        job,
        sal,
        comm,
        sal + IFNULL(comm, 0)  AS  total_sal
FROM    emp;
Q115% 증가 급여 및 증가액
SELECT  empno,
        ename,
        job,
        sal,
        sal + sal * 0.15   AS  NewSalary,
        sal * 0.15         AS  Increase
FROM    emp;
Q2입사일로부터 현재까지 년수 및 급여 총계
SELECT  ename,
        hiredate,
        TIMESTAMPDIFF(YEAR, hiredate, SYSDATE())          AS  years_work,
        sal,
        sal * 12 * TIMESTAMPDIFF(YEAR, hiredate, SYSDATE())  AS  total_sal
FROM    emp;
Q3입사한 요일 출력 (두 가지 방법)
-- 방법1: DAYNAME 함수 (영문 요일)
SELECT  ename, job, hiredate, DAYNAME(hiredate)  AS  day_of_week
FROM    emp;

-- 방법2: CASE + DAYOFWEEK (한글 요일)
SELECT  ename, job, hiredate,
        CASE DAYOFWEEK(hiredate)
            WHEN 1 THEN '일'  WHEN 2 THEN '월'  WHEN 3 THEN '화'
            WHEN 4 THEN '수'  WHEN 5 THEN '목'  WHEN 6 THEN '금'
            WHEN 7 THEN '토'
        END  AS  day_of_week
FROM    emp;
Q4급여를 15자리로, 좌측 빈 곳에 * 대치
SELECT  ename,
        LPAD(sal, 15, '*')  AS  sal
FROM    emp;

모든 문제를 풀어본 후 아래 버튼으로 정답을 확인하세요.

Q12003년에 입사한 사원
SELECT  employee_id, last_name, hire_date
FROM    employees
WHERE   YEAR(hire_date) = 2003;
Q2특정 업무코드 사원 조회 (IN)
SELECT  *
FROM    employees
WHERE   job_id IN ('FI_ACCOUNT', 'FI_MGR', 'ST_MAN', 'ST_CLERK');
Q3커미션을 받는 사원 (IS NOT NULL)
SELECT  employee_id, last_name, hire_date, commission_pct
FROM    employees
WHERE   commission_pct IS NOT NULL;
Q4부서 구분 출력 (CASE)
SELECT  job_id,
        CASE job_id
            WHEN 'ST_MAN'   THEN '판매부서'
            WHEN 'ST_CLERK' THEN '판매부서'
            ELSE '그 외 부서'
        END  AS  dept_type
FROM    employees
WHERE   job_id LIKE 'S%';
Q515% 증가 급여 및 증가액
SELECT  employee_id,
        last_name,
        job_id,
        salary,
        salary + salary * 0.15   AS  NewSalary,
        salary * 0.15            AS  Increase
